# Kanban-доска

React-приложение для ведения задач в формате канбана. В проекте реализована следующая функциональность:

1. Вход и авторизация;
2. Добавление новой задачи авторизованным пользователем с возможностью выбора категории;
3. Отображение добавленных задач по статусам выполнения;
4. Редактирование задач текущим пользователем (доступно для редактирования: описание задачи, крайний срок выполнения, статус задачи);
5. Дополнительно реализованы оповещения о входе и выходе с использованием библиотеки React-Toastify.

## Технологии

В проекте использованы технологии: JavaScript, HTML, React + Vite, Git, REST API.
